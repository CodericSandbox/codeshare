Codeshare
=========

## Description
Codeshare lets you create collaborative docs with CodeMirror and ShareJs.  

## Installation
```
   $ git clone git://github.com/tarr11/codeshare

   $ npm install
```
## Running
   node bin/codeshare.js

   open 2 browser windows to http://localhost:8000
   
   start typing!
